# 七鱼回复助手
## 此工具根据张立烨编写的回复脚本为基础 加入了自动注入和快捷选择服务小记类型

下载 “七鱼助手.exe” 并运行

        https://github.com/jonaszhang91/qiyuHelper/raw/refs/heads/main/%E4%B8%83%E9%B1%BC%E5%8A%A9%E6%89%8B.exe

- 本项目为搭建框架后AI 完善，部分备注非常抽象，有发现代码看不懂的可以找我 我来完善备注信息
- 首次开启需要选择 七鱼路径
- 脚本内容自动更新
- 右键图标可以选择快捷选择服务小记类型
- 快捷填写现在暂时是4个 ：“POS问题”“刷卡机问题”“打印机问题”“其他” 如需增加 请飞书，微信或者当面找我
